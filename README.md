# Signal Processing

Códigos desenvolvidos para a disciplina de Processamento de Sinais e Sistemas Mecânicos Lineares. Cada análise fica em uma pasta própria, com README, código e notebook de exemplo.

## Projetos

| Pasta | Descrição |
|---|---|
| [`janelamento_fft_hilbert/`](janelamento_fft_hilbert/) | Análise de sinais experimentais: janelamento (Hanning, Hamming, ...), espectro de amplitude (FFT) e transformada de Hilbert (amplitude, fase e frequência instantâneas) |

## Como usar

Entre na pasta do projeto desejado e siga o README dela. Em geral:

```bash
cd janelamento_fft_hilbert
pip install -r requirements.txt
```

e abra o notebook de exemplo da pasta.
